# Macro Tracker Frontend

Next.js frontend for the Macro Tracker application with mobile-responsive design.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Features

- **Onboarding Form**: Collect user profile information
- **Travel Intensity Selector**: Choose activity level based on travel patterns
- **Macro Dashboard**: View daily macro targets and progress
- **Circular Progress Bars**: Visual representation of macro consumption
- **Mobile Responsive**: Optimized for travelers on the go

## Backend Integration

Make sure the FastAPI backend is running on `http://localhost:8000` before using the app.
